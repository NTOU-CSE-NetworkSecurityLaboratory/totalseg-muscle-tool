# TotalSeg WebView 介面（使用者版）

這是 TotalSeg 的網頁風格桌面介面。

## 怎麼啟動

Windows 直接雙擊：

`START WebView Tailwind.bat`

## 可以做什麼

- 掃描 DICOM 資料夾
- 批次執行分割（Start / Stop）
- 比對 AI 與人工標註（Compare）
- 在畫面看執行紀錄（Logs）
- 遇到授權問題可直接在視窗內補 key 後重試

## 小提醒

- 視窗關閉時，正在跑的任務會一起停止。
- 執行紀錄會自動存到資料夾內的 `totalseg_batch_logs`。
- 本工具用於研究與流程輔助，不可用於臨床診斷。
